### 1. 股票标的推荐格式

```markdown

## 基本信息
- **代码**: 600000.SH / 000001.SZ
- **名称**: 浦发银行
- **推荐日期**: 2026-03-08
- **推荐价格**: 10.50元
- **止损位**: 9.80元 (-6.7%)
- **目标位**: 12.00元 (+14.3%)
- **仓位建议**: 5-10%

## 推荐逻辑
1. **技术面**: 突破年线，MACD金叉
2. **基本面**: 股息率5%+，估值低位
3. **催化剂**: 政策利好银行板块

## 操作建议
- 入场条件: 回踩10.30元不破
- 持有周期: 2-4周
- 退出条件: 跌破止损或达到目标位

## 跟踪记录
| 日期 | 价格 | 涨跌幅 | 备注 |
|------|------|--------|------|
| 2026-03-08 | 10.50 | 0% | 建仓 |

## 相关标的
- 联动: 招商银行、平安银行
- 对冲: 黄金ETF

## 标签
#银行 #高股息 #技术突破
```
